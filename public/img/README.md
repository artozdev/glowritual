# Images de la landing

Dépose ici tes deux photos pour le comparateur Avant / Après :

- `before.jpg` → la photo **Avant** (peau à sublimer)
- `after.jpg` → la photo **Après** (peau nette et lumineuse)

Tant qu'elles sont absentes, un repli Unsplash s'affiche automatiquement.
Formats acceptés : `.jpg` / `.png` (garde le même nom de fichier).
